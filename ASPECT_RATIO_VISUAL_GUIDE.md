## 🎯 Three Ways to Use It

### 1. Quick Command Line
```bash
curl -X POST http://localhost:5000/analyze \
  -F "youtube_url=https://youtube.com/watch?v=VIDEO_ID" \
  -F "ratio=9:16" \
  -F "padding_color=black"

# Response: 4 TikTok-ready clips
```

### 2. Interactive Demo Page
```
1. Open: http://localhost:5000/templates/aspect-ratio-generator.html
2. Paste YouTube URL
3. Select aspect ratio
4. Click "Generate"
5. Download clips
```

### 3. Integrated in Your Form
```html
<form method="POST" action="/analyze">
    <input type="text" name="youtube_url" required>
    <select name="ratio">
        <option value="9:16">TikTok (9:16)</option>
        <option value="16:9">YouTube (16:9)</option>
        <option value="1:1">Instagram (1:1)</option>
    </select>
    <select name="padding_color">
        <option value="black">Black</option>
        <option value="white">White</option>
        <option value="blur">Blur</option>
    </select>
    <button type="submit">Generate</button>
</form>
```
