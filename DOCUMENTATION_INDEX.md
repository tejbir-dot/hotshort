## API Quick Reference

### Endpoint
```
POST /analyze
```

### Parameters
```
youtube_url    (required) - YouTube URL
ratio         (optional) - "16:9", "9:16", "1:1", "21:9", "4:3", "native"
padding_color (optional) - "black", "white", "blur"
```

### Example cURL
```bash
curl -X POST http://localhost:5000/analyze \
  -F "youtube_url=https://youtube.com/watch?v=..." \
  -F "ratio=9:16" \
  -F "padding_color=black"
```

### Response
```json
[{
  "title": "Viral Moment #0",
  "clip_url": "/static/outputs/clip_0_23.4_49.8.mp4",
  "start": 23.4,
  "end": 49.8,
  "score": 0.87,
  "ratio": "9:16"
}]
```

```
Per Clip:
├─ Extract: 0.5-2s
└─ Apply ratio: 5-20s
└─ Total: ~10-15s average

For 4 Clips:
├─ Black padding: 20-40 seconds
├─ White padding: 24-44 seconds
├─ Blur padding: 40-80 seconds
└─ Total time: 30-70 seconds

Best Performance:
├─ Use: Black padding
├─ Speed: 5-10s per clip
├─ Total: 20-40s for 4 clips
```
